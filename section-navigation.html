Section Navigation

Section navigation should be used when a section of a site has a substantial number of pages and requires some sort of generic navigation in order for users to be able to find all content. A section that would be a good candidate for this component is very content heavy and has many different types of pages with varying layouts or formats. A section that has lots of content, but all of the same type or format should instead have a listing page or some other UX to get users to the content.

Example

Tableau Starter Kit
  • Author viz
  • Interact with a viz
    • Tableau Online
    • Tableau server
  • Publish a viz
    • Tableau Online
    • Tableau server
  • Administer Tableau
