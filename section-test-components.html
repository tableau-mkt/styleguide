Test Components

Components that we aren't yet fully committed to and are testing out via A/B testing or proving out in some other manner. All components in this section should eventually find a permanent home in an existing section or be removed if they don't pan out. Components in this section should have a created date and an author in the description so it's easier to track down their intended use and figure out how old they are.

Do not use any of these components unless for a specific test. They are at risk of being deleted or refactored.

News Banner

A News Banner displays content to promote an important announcement to the user. A News Banner is displayed the first time a user visits the page and then stores a cookie as soon as they dismiss the message to not show the same message to the user again. Individual messages are identiified by their id attribute so it's important to make sure different messages have a unique ID.

Due to the nature of fixed positioned elements, the examples below don't function properly. This component is meant to be placed at the very top of a page in order to be displayed properly.

News Interstitial

An interstitial display of content to promote an important announcement to the user. An interstitial is displayed the first time a user visits the page and then stores a cookie as soon as they dismiss the message to not show the message same message to the user again. Individual messages are identiified by their id attribute so it's important to make sure different messages have a unique ID. The background image used for the interstitial should be 1000px by 500px.

Launch the interstitial demo by clicking one of the following links and refreshing:

  • Dark Modifier
