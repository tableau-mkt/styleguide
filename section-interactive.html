Wrapper Options:

  • data-tabs-scroll="[wrapper|content]" - (optional) - Smooth-scroll to either the top of the tabs wrapper or the target content when tabs are clicked.
  • data-tabs-duration="500" - (optional) - override the default duration of the JS height animation. Useful for canceling out the transition by setting to 0

Content Reveal

Simple tool to make content hidden and revealed when a trigger is clicked. The trigger element can be anywhere on the page and can have the following options set as data attributes:

Trigger Options:

  • data-reveal-target="[CONTENT ID]" - (required) - The ID of the container around the content to be revealed
  • data-reveal-hide-text="[REVEAL HIDE TEXT]" - (optional) - Text to swap into the trigger link while the reveal is in the open state.
  • data-reveal-close-link="false" - (optional) - Set this to disable the default close link within the reveal content container
  • data-reveal-scroll="[trigger|target]" - (optional) - Smooth-scroll to either the top of the trigger or the target element when the content is revealed.
  • data-reveal-media="video" - (optional) - Enable play/pause controle when opening/closing reveal content that contains a brightcove video.

The .reveal__content container must have an id attribute that matches the data-reveal-target.

Example

Reveal Content

<div id="curtain">
  <p>
      <a href="#" class="reveal__trigger cta" data-reveal-target="content-container" data-reveal-media="markup" data-reveal-type="false" data-reveal-hide-text="Hide Content" data-reveal-curtain="curtain">Reveal Content</a>
  </p>
</div>

Scroll Reveal

Allows you to fade content into view in a number of ways. The default is a simple fade-in once the element is at least 20% into the viewport. Can be customized extensively with options via data attributes or use the preset modifiers. This component uses the scrollreveal plugin for all interactions with several custom features added including support for options via data attributes and custom sequencing of reveals within a container.

Examples

Default styling

Scroll Reveal Customization

Scroll Reveals can be used to liven up a page and make the content more engaging. The idea is to draw some attention to elements as they come on the page. This works great for bringing images onto the page ro revealing short content blurbs.

Scroll reveals can be tweaked and fine-tuned in markup by setting data attributes for all of the options that the Scroll Reveal plugin supports. Refer to the plugin documentation for examples and descriptions. Here's some examples:

data-origin="top"

data-distance="200px"

data-duration="1000"

data-delay="500"

data-opacity="0.5"

data-scale="0.5"

data-mobile="false"

data-reset="true"

data-useDelay="once"

data-viewFactor="0.8"

Sequenced Scroll Reveal

Content can be revealed on the page in a sequence with an interval between subsequent reveals by wrapping the items in a scroll-reveal__sequence-container container and giving each revealing element a scroll-reveal--sequenced class. An optional data-sequence-interval="200" attribute on the sequence container will alter the interval between reveals within the container. If not set, this will default to 100ms. Additionally, a data-sequence-delay="200" attribute can be added to delay the start of the sequence. If not set, it defaults to 0.
