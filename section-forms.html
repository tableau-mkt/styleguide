Contextual Search

Contextual search form for section-specific searches.

Upon submitting (e.g., pressing ENTER or clicking the search submit icon) an intermediate search results list is shown in the context of the page (provided via AJAX).

More results text, at the bottom of the intermediate search results list, allows you to continue to the full results page.

Down/up arrow keys allow you to select a results row, and clicking outside the search component will close the search results.

Example

3 MIN
TOPICS > CONNECTING TO DATA
Google Sheets API
2 MIN

Good Storytelling with Data
3 MIN

      <button class="content-search__button content-search__reset" type="reset" name="exampleSearchReset"><i class="icon icon--close-window-style2"></i></button>
    </form>
