Contextual Search

Contextual search form for section-specific searches.

Upon submitting (e.g., pressing ENTER or clicking the search submit icon) an intermediate search results list is shown in the context of the page (provided via AJAX).

More results text, at the bottom of the intermediate search results list, allows you to continue to the full results page.

Down/up arrow keys allow you to select a results row, and clicking outside the search component will close the search results.

Example

Google Analytics
3 MIN
TOPICS > CONNECTING TO DATA
Google Sheets API
2 MIN
TOPICS > CONNECTING TO DATA
Good Storytelling with Data
3 MIN
TOPICS > DASHBOARDS & STORIES
 65 results for "goo" sitewide

Markup

<div class="contextual-search">
  <div class="contextual-search__ui">
    <form class="contextual-search__form" autocomplete="off">
      <input class="contextual-search__input" type="search" name="exampleSearch" placeholder="Search this section">
      <button class="contextual-search__button contextual-search__submit" type="submit" name="exampleSearchSubmit"><i class="icon icon--search"></i></button>
      <button class="contextual-search__button contextual-search__reset" type="reset" name="exampleSearchReset"><i class="icon icon--nic-cage"></i></button>
    </form>
    <div class="contextual-search__results">
      <a href="#row0" class="contextual-search__results-row">
        <div class="contextual-search__title">Google Analytics</div>
