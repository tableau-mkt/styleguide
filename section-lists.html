Content Lists

Lists of content that generally have a title, thumbnail and/or a description and link through to the full version of the content.

Standard Unordered List

A standard unordered list with bullets. Gray background applied in order to make sure light colored bullets show up.

Examples

Default styling
  • For anyone.
  • Interactive data visualizations and dashboards.
  • Works with 44 data sources.

Light colored bullets.

.list-negative

  • For anyone.
  • Interactive data visualizations and dashboards.
  • Works with 44 data sources.

Markup

<ul class="[modifier class] ">
    <li >
      For anyone.
    </li>
    <li >
      Interactive data visualizations and dashboards.
    </li>
    <li >
      Works with 44 data sources.
    </li>
</ul>

Link List

A simple unordered list of links. Several helper modifiers provided.

Examples

Default styling
  • See it in Action
  • Get Free Trial

Plain list (no bullets)

.link-list--plain

  • See it in Action
  • Get Free Trial

Inline list items

.link-list--inline

  • See it in Action
  • Get Free Trial

Markup

<ul class="link-list [modifier class] ">
    <li class="active">
        <a href="http://www.tableau.com"  class="">See it in Action</a>
    </li>
    <li class="download">
        <a href="http://www.google.com" target="_blank" class="">Get Free Trial</a>
    </li>
</ul>
