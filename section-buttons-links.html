Toggle Links

A widget to indicate toggling between two different options.

Note: The interactivity of this component works by toggling an is-active class between the options. This is not done automatically by this component and must be used in conjunction with another component, such as Content Tabs, that manages the classes.

Examples

Default styling
Option 1 Another Option

wider links

toggle-links--wide

Option 1 Another Option

Markup

<div class="toggle-links [modifier class]">
    <a href="#option1" class="toggle-links__option is-active">Option 1</a>
    <a href="#option2" class="toggle-links__option">Another Option</a>
</div>

Topic Link

Links to a topic that a piece of content was tagged with. Meant to be used on a a full content view page as the primary topic that the piece of content is tagged with.

Example

"Back To" link

A link displayed at the top of a page that links back to a parent page. For example, a blog article page would have a "Back to Blog" link that links back to the main blog overview page.

Example

Back to [Section]

Markup

<div class="back-to-link">
  <a href="#" class="link  link link--arrow-before" >Back to [Section]</a>
</div>
