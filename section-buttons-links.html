Topic Link

Links to a topic that a piece of content was tagged with. Meant to be used on a a full content view page as the primary topic that the piece of content is tagged with.

Example

Nicolas Cage Movies

Markup

  <a class="topic-link" href="#">Nicolas Cage Movies</a>

Anchor Link

Acts as a container to support links to this specific spot on the page.

Example

Markup

<a id="anchor-link" class="anchor-link"></a>

"Back To" link

A link displayed at the top of a page that links back to a parent page. For example, a blog article page would have a "Back to Blog" link that links back to the main blog overview page.

Example

Back to [Section]

Markup

<div class="back-to-link">
  <a href="#" class="link  link link--arrow-before">Back to [Section]</a>
</div>
