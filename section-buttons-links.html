Secondary CTA buttons (Stroked Orange) should be used when multiple page actions are present, or somewhat important page actions are required.

Tertiary CTA buttons (Stroked alternate) should be used when a utility action, or an action not directly related to the page is required. When hovering over a Stroked button the copy is underlined.

Topic Link

Links to a topic that a piece of content was tagged with. Meant to be used on a a full content view page as the primary topic that the piece of content is tagged with.

Example

Nicolas Cage Movies

Markup

  <a class="topic-link" href="#">Nicolas Cage Movies</a>

Text-based links, unlike CTA buttons, don't have a colored box around them, although they can have an accompanying icon similar to CTAs. All links underline on hover.
