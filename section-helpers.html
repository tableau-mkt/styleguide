CSS Helpers

General helper classes that can be used to overcome various common issues. Classes here are meant to be either added to existing elements or to <div> or <span> wrappers around an element via a class="helper-class" attribute.

Hiding Elements

Different methods of hiding elements in idfferent contexts

hide

Hide the element from being displayed in all breakpoints

mobile-hide

Hide the element from display only on the mobile breakpoint

mobile-show

Show element only on the mobile breakpoint

tablet-hide

Hide the element from display only on the tablet breakpoint

desktop-show

Show element only on the desktop breakpoint

desktop-hide

Hide the element from display only on the desktop breakpoint

tablet-show

Show element only on the tablet and mobile breakpoint

reveal-hidden

Set an element to an initially hidden state but to be revealed later.

Padding & Margins

Add margin/padding to a component or element.The list of classes below is not exhaustive, but shows examples of all of the different variations of these helper classes. Each class is broken into 3 different fragments.

  • Property - { margin | padding } - The property to adjust
  • Side(s) - { top | right | bottom | left | horz | vert } - The side(s) to apply the padding or margin to. When a direction is left out the spacing applies to all sides.
  • Multiplier - { none | one | two | three | four } - The number of "em" units to apply

The classes follow the format of property-side-multiplier

margin-none

Remove all margins

margin-top-none

Remove top margin

margin-one

Add 1em of margin to all sides

margin-left-two

Add 2em of margin to the left

margin-horz-three

Add 3em of margin to both the right and left

padding-none

Remove all padding

padding-bottom-one

Add 1em of padding to the bottom

padding-right-one

Add 1em of padding to the right

padding-vert-two

Add 2em of padding to both the top and bottom

Centering Elements

Different helpers for horizontally and/or vertically centering different types of elements.

element-centered

Center a block-level element using auto margins.

inline-centered

Center an inline element.

vertical-align-middle

Vertically aligns an element to the middle. Works well with display table-cell

vertical-centered

Vertically and horizontally center an element. This assumes the parent has some kind of defined height and has position: relative;

vertical-centered-flex

Vertically center an element using flexbox. Be aware this is unsupported in older browsers. Apply to the parent wrapper of elements that need vertical centering.

absolute-centered

Vertically and horizontally center an element. This assumes the parent has some kind of defined height and has position: relative;

flex-justified

Horizontally evenly space out elements

List tweaks

Modifying default styles of an HTML list

plain-list

inline-list

Same as .plain-list, but also makes list items inline-block

Touch Scroll Handling

Handlng for embedded content on touch devices and viewports narrower than the embed content. Allows the content of the wrapper with this class to scroll horizontally while not affecting the responsive layout of the rest of the page. Useful wrapper for Tableau embeds with a fixed width. Also automatically horizontally centers Tableau embeds on wider viewports.

overflow-touch-scroll

Allow wide embeds to scroll horizontally on narrow viewports
